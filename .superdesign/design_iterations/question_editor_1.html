Design Notes

  • Collapsed by default — Helper text, video, and skip logic hidden behind "More options" toggle
  • Badge count — Shows number of configured advanced options when collapsed (e.g., "1" if skip logic is set)
  • Auto-expand — If helper text, video URL, or branching rules exist, section starts expanded
  • AI-generated tag — Skip logic rules from voice generation show purple "AI generated" badge
  • Visual hierarchy — Question text + type + options are always visible (the essentials). Everything else is progressive disclosure.
  • Density reduction — ~40% less vertical space per question card in collapsed state
