Quick start

Connect in under a minute.

Settings → Connectors

1. Open Claude Desktop → Settings → Connectors

2. Click "Add custom connector"

3. Paste the server URL:

https://mcp.agentcrm.dev/sse

4. Complete the OAuth sign-in when prompted

5. Start chatting — AgentCRM tools appear automatically

No install, no config files. Works on Free, Pro, Max, Team, and Enterprise plans. Also works on Claude mobile.

Settings → Developer → Edit Config 
{
  "mcpServers": {
    "agentcrm": {
      "command": "npx",
      "args": ["@agentcrm/mcp-server"],
      "env": {
        "UPSIGHT_API_KEY": "upsk_your-api-key"
      }
    }
  }
}

Go to Settings → Developer → Edit Config and paste the above. Credentials stay local — never hit the network.

Settings → Connectors → Create

1. In ChatGPT, go to Settings → Connectors

2. Click "Create" to add a new connector

3. Enter a name (e.g. "AgentCRM") and paste the server URL:

https://mcp.agentcrm.dev/mcp

4. Complete the OAuth flow to authorize access

5. AgentCRM tools now appear in both Chat and Deep Research modes

Requires ChatGPT Pro, Team, Enterprise, or Edu plan. Enable Developer Mode in Advanced settings for full tool access.

Terminal

Option A: Install via ClawHub (recommended)

$ clawhub install agentcrm/mcp-server

Option B: Manual configuration

$ openclaw config set mcpServers.agentcrm.command "npx"
$ openclaw config set mcpServers.agentcrm.args '["@agentcrm/mcp-server"]'
$ openclaw config set mcpServers.agentcrm.env.UPSIGHT_API_KEY "upsk_your-api-key"

Config stored at ~/.openclaw/openclaw.json. Works with all OpenClaw messaging bridges — WhatsApp, Telegram, Discord, and more.

User journeys

Your path from install to insight.

Claude Desktop User

1

Authorize — Complete OAuth (remote) or paste your API key (local). One time.
