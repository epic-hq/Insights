## User Flows

### Stakeholder Engagement Flow
```
1. CREATE STAKEHOLDER PROFILE
   └─ Contact details
   └─ Interests, concerns
   └─ Organization/community
   └─ Auto-sync across projects

2. LOG INTERACTIONS
   └─ Meeting, call, email, event
   └─ Attach sentiment (neg/neutral/pos)
   └─ Tag issues raised
   └─ Assign follow-ups

3. TRACK ISSUES & COMMITMENTS
   └─ Create issues from interactions
   └─ Set priority/impact
   └─ Track resolution timeline
   └─ Log commitments made
   └─ Set recurring follow-ups

4. REPORT & ANALYZE
   └─ Generate stakeholder reports
   └─ View engagement history
   └─ KPI dashboards
   └─ Export CSV/PDF
```

## Competitive Position

### Their Strengths

1. **Canadian Market**: Strong in resources/energy sector
2. **Indigenous Focus**: Specific workflows for indigenous relations
3. **Integration Rich**: ArcGIS, Outlook, Zapier, API
4. **Sentiment Tracking**: Built-in attitude monitoring
5. **Cross-Project Sync**: Stakeholders persist across projects
6. **Employee-Owned**: Aligned incentives, stable company

### Their Weaknesses

1. **No Voice**: Cannot capture or transcribe interviews
2. **No AI**: All analysis is manual
3. **Manual Entry**: Everything requires human input
4. **No Synthesis**: Cross-stakeholder patterns manual
5. **No Deliverables**: Doesn't generate consulting outputs
6. **Price Point**: $995+/month is high for SMB

## Threat Assessment

**Threat Level**: LOW-MEDIUM

**Why Low-Medium**:
- Different primary market (resources vs. consulting)
- No AI/voice direction visible
- Price point higher than SMB consulting
- Strong in specific verticals (indigenous, resources)

**Risk Scenario**:
- If they add AI/voice and target consulting, could compete
- Strong parent company (Silvacom) could fund expansion

**Watch For**:
- AI feature announcements
- Voice interview capabilities
- Consulting vertical marketing
- Pricing changes (lower entry tier)
