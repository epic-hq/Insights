## Our Positioning

**UpSight's Key Differentiators:**

1. **Voice-First Setup** - Natural language onboarding, not enterprise forms
2. **Intelligent Suggestions** - Context-aware AI recommendations
3. **Consumer-Grade UX** - Typeform-style simplicity with professional features
4. **Speed to Insight** - Setup in minutes, not hours

## Adding New Competitors

When adding a new competitor analysis:

1. Create `{company-name}.md` in this directory
2. Include:
   - Overview and positioning
   - Key products/features
   - Strengths and weaknesses
   - How we beat them (specific tactics)
   - Marketing tactics
   - Feature roadmap implications
3. Update this README with the new entry
