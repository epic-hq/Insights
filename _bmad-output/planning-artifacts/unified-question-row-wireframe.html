What changed (v5)

Drop-off is integrated. Each question row has a thin vertical bar on the right edge — tall + green = healthy, short + red = cliff. Percentage shows on hover. No separate funnel card eating space.

Hover results are inline. Hover any question and a slim panel expands below it showing response data right there. Select questions show answer distribution with mini bars. Likert shows histogram + average. Open text shows word count, time, skip rate. No popover that clips off-screen.

Q4 is always-visible (red background + stats shown) because it's the problem child. Uppy references it in chat.

Branching visualization

Left border accent on questions that have branch rules — color-coded (indigo for skip-logic, amber for conditional follow-ups).

Branch annotation row sits directly below the parent question showing the rule in plain language: condition → action. Right-aligned shows what % of respondents took that path (when data exists).

Conditional children (Q4a) are indented with matching left border and a "conditional" tag. They visually nest under their parent.

Two branch types shown:

1. Skip logic (indigo) — "If None → skip to Q5" jumps ahead, skipping Q3–Q4

2. Conditional show (amber) — "If score ≤ 6 → show Q4a" reveals a hidden follow-up

The fork icon in the question row meta tells you at a glance "this one branches." Click the branch annotation to edit the rule in the drawer.
